Warclight

Warclight is a Project Blacklight based Rails engine that supports the discovery of web archives held in the WARC and ARC formats. It allows faceted full-text search, record view, and other advanced discovery options.

Want to know more? You can visit our example engine here, or our GitHub repository here.
